DCMA Entry Workflow and Tool Usage

This document outlines the workflow for submitting a DCMA entry and the usage of tools by the site manager:

  1. The site manager receives a DMCA claim via the dedicated email. They fill in the following information:

    • Date Request Received
    • Date Notice Sent
    • Date Dispute Received
    • Notes
    • Status (Active/Processed)

    Before submitting the DCMA entry, the site manager hides contested reviews using the dropdown and hide review button. The site manager then submits the DCMA entry.

  2. After the claim is disputed (taken down or reposted), the site manager selects the previous complaint from the dropdown and marks it as processed.

  3. If the claim is disputed and processed, the site manager can use the dropdown to select the previously hidden review and press the restore contested review button to restore it.
